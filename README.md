# 題目(加密字符串)
密碼對照表的第一行為明碼行，放置大寫字母

第二行為依照下列規則產生的密碼行：

給定一個單詞，將單字中所有字元轉為大寫字母，對於單字中重複出現的字母，保留第一次出現的，刪除之後重複出現的該字元。
用剩餘字母組成秘鑰從密碼行的起始位置放置；再用未在秘鑰中出現的其他大寫字母按字母表順序依序填入密碼行剩餘位置。
兩筆輸入

1.輸入一個字串作為密鑰
2.輸入一個需要加密的字串
